# 🚌 College Bus Tracking System

Real-time college bus tracking system built with Flutter & Firebase.

## 🔥 Features
- 📍 Live GPS tracking (Driver → Firebase → Student App)
- 🗺️ Real-time map with bus location
- 🔔 Push notifications
- 📊 Trip history
- 🔍 Bus search
- 👤 Student profile

## 🛠️ Tech Stack
| Layer | Technology |
|-------|-----------|
| Student App | Flutter (Android) |
| Database | Firebase Realtime DB |
| Driver Tracking | Browser GPS API |
| Maps | Custom Map UI |
| Notifications | Firebase FCM |

## 🚀 Setup
1. Clone the repo
2. Add `google-services.json` to `android/app/`
3. Run `flutter pub get`
4. Run `flutter run`
